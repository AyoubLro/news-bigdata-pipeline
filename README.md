# news-bigdata-pipeline
🚀 End-to-End Big Data News Analytics Pipeline built using modern Data Engineering technologies.

This project collects real-time news articles from multiple online sources, processes and analyzes the data through a scalable pipeline architecture, and visualizes insights using an interactive dashboard.

🔧 Technologies Used:
• Python
• Apache Kafka
• Apache Airflow
• MinIO Data Lake
• SQLite
• Streamlit
• Docker

📌 Features:
✔ Real-time data streaming with Kafka
✔ Automated workflows with Airflow
✔ Bronze / Silver / Gold Data Lake architecture
✔ Data cleaning and transformation
✔ News analytics and visualization dashboard
✔ Containerized infrastructure with Docker

The project demonstrates a complete Data Engineering workflow from data collection to storage, orchestration, analysis, and visualization.
